1. Basic Views (UI Components)

TextView - Menampilkan Text

<TextView
    android:id="@+id/textView1"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hello Android!"

Atribut Penting:

  • android:text: Text yang ditampilkan

4. Handling User Interactions

Click Events di Java

public class MainActivity extends AppCompatActivity {
    private Button submitButton;
    private EditText nameInput;
    private TextView resultText;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        
        // Initialize views
        submitButton = findViewById(R.id.submitButton);
        nameInput = findViewById(R.id.nameInput);
        resultText = findViewById(R.id.resultText);
        
        // Set click listener
        submitButton.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                handleSubmitClick();
            }
        });
    }
    
    private void handleSubmitClick() {
        String name = nameInput.getText().toString().trim();
        
        if (name.isEmpty()) {
            nameInput.setError("Nama tidak boleh kosong");
            return;
        }
        
        resultText.setText("Hello, " + name + "!");
        resultText.setVisibility(View.VISIBLE);
    }
}

Lambda Expression (Java 8+)

// Cara modern dengan lambda
submitButton.setOnClickListener(v -> handleSubmitClick());

// Multiple buttons
button1.setOnClickListener(v -> showToast("Button 1 clicked"));
button2.setOnClickListener(v -> showToast("Button 2 clicked"));

private void showToast(String message) {
    Toast.makeText(this, message, Toast.LENGTH_SHORT).show();
}

LoginActivity.java

public class LoginActivity extends AppCompatActivity {
    private EditText emailInput, passwordInput;
    private Button loginButton;
    private TextView registerLink;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_login);
        
        initViews();
        setupClickListeners();
    }
    
    private void initViews() {
        emailInput = findViewById(R.id.emailInput);
        passwordInput = findViewById(R.id.passwordInput);
        loginButton = findViewById(R.id.loginButton);
        registerLink = findViewById(R.id.registerLink);
    }
    
    private void setupClickListeners() {
        loginButton.setOnClickListener(v -> handleLogin());
        registerLink.setOnClickListener(v -> openRegisterActivity());
    }
    
    private void handleLogin() {
        String email = emailInput.getText().toString().trim();
        String password = passwordInput.getText().toString().trim();
        
        // Validation
        if (email.isEmpty()) {
            emailInput.setError("Email tidak boleh kosong");
            emailInput.requestFocus();
            return;
        }
        
        if (!android.util.Patterns.EMAIL_ADDRESS.matcher(email).matches()) {
            emailInput.setError("Format email tidak valid");
            emailInput.requestFocus();
            return;
        }
        
        if (password.isEmpty()) {
            passwordInput.setError("Password tidak boleh kosong");
            passwordInput.requestFocus();
            return;
        }
        
        if (password.length() < 6) {
            passwordInput.setError("Password minimal 6 karakter");
            passwordInput.requestFocus();
            return;
        }
        
        // Simulate login
        showToast("Login berhasil!");
        // TODO: Navigate to main activity
    }
    
    private void openRegisterActivity() {
        showToast("Fitur registrasi akan segera hadir");
    }
    
    private void showToast(String message) {
        Toast.makeText(this, message, Toast.LENGTH_SHORT).show();
    }
}

üîë Poin Penting

  • View adalah komponen UI dasar di Android
  • Layout mengatur posisi dan ukuran Views
  • ConstraintLayout adalah layout modern yang direkomendasikan
  • Resources (colors, strings, dimens) membuat kode lebih maintainable
  • Click listeners menangani interaksi user
  • Validation penting untuk user experience yang baik

‚ùå Common Errors

1. "findViewById returns null"

Penyebab: ID tidak ditemukan atau salah layout
Solusi: Pastikan ID benar dan setContentView() sudah dipanggil

2. "App crashes on button click"

Penyebab: Listener tidak di-set atau null pointer
Solusi: Pastikan findViewById dan setOnClickListener sudah benar

3. Layout tidak sesuai harapan

Penyebab: Constraint atau layout parameter salah
Solusi: Gunakan Layout Inspector untuk debug layout

4. EditText tidak bisa di-input

Penyebab: focusable="false" atau tidak ada inputType
Solusi: Set inputType yang sesuai
